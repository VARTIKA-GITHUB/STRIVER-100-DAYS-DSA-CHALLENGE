# STRIVER-100-DAYS-DSA-CHALLENGE

Welcome to my Striver DSA Challenge repository!

I have taken on a challenge to solve **3 Data Structures and Algorithms (DSA) problems every day** from Striver's DSA Sheet to strengthen my problem-solving skills, improve coding consistency, and prepare for technical interviews.

## 📌 Challenge Goals
- Solve 3 DSA problems daily.
- Build strong problem-solving fundamentals.
- Improve coding speed and logical thinking.
- Maintain consistency through public accountability.
- Track progress throughout the challenge.

## 🛠️ Languages Used
- CPP

## 📂 Repository Structure
Each day's solutions are organized by date/problem name and include:
- Problem statement link
- Approach/logic
- CPP solution
- Time & Space Complexity

## 🎯 Current Focus
- Arrays
- Strings
- Linked Lists
- Stacks & Queues
- Trees
- Graphs
- Dynamic Programming
- Interview Patterns

## 📈 Progress
This repository will be updated daily with new solutions and learnings.
